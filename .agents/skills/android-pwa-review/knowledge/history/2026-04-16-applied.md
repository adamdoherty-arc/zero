## Changes shipped

### 1. Reference Video Inbox detail sheet mutations (+2.0 pts)

**File**: [frontend/src/pages/MobileVideosPage.tsx](frontend/src/pages/MobileVideosPage.tsx)

Added a 2-column action grid to the bottom sheet that exposes every mutation a
reviewer needs on a phone:

- **Assign / Reassign** (`UserPlus`) opens an in-sheet character picker with
  search, reusing `useCharacters({ status: 'active' })` and marking the current
  assignment with a check. Calls `useAssignReferenceVideoCharacter`.
- **Promote** (`Sparkles`) opens an in-sheet form pre-filled from
  `video.proposed_character` (name / universe / franchise / description) and
  calls `usePromoteReferenceVideoToCharacter`.
- **Apply facts** (`BookOpen`) opens a checklist of `video.extracted_facts` with
  all selected by default; submits selected indexes to
  `useApplyReferenceVideoFacts`. Disabled with a tooltip when no character is
  assigned or no facts are extracted. Shows `applied_fact_count` in the facts
  header.
- **Retry** (`RefreshCw`) appears instead of Delete when `status === 'failed'`;
  calls `useRetryReferenceVideo`. Delete remains reachable beneath the grid in
  that variant.
- **Delete** (`Trash2`) shows an inline confirm row with Cancel / Delete buttons
  (no nested modal). Calls `useDeleteReferenceVideo` and closes the sheet on
  success.

All action buttons are `min-h-[48px]`, subsheets use `min-h-[44px]` inputs, and
the sheet keeps `paddingBottom: env(safe-area-inset-bottom)`. An "Assigned"
badge is surfaced in the header when `video.character_id` is set.

### 2. SharePage safe-area-inset-top (+0.75 pts)

**File**: [frontend/src/pages/SharePage.tsx:111](frontend/src/pages/SharePage.tsx#L111)

The page container now uses:

```tsx
style={{
    paddingTop: 'max(1rem, env(safe-area-inset-top))',
    paddingBottom: 'max(1rem, env(safe-area-inset-bottom))',
}}
```

SharePage runs outside `MobileLayout` (it's a direct target of the share sheet),
so it has to carry its own safe-area padding. Notches and hole-punch cameras no
longer clip the "Send to Zero" header.
