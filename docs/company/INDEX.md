# ADA AI LLC Company Docs

Zero is the active software home for ADA AI LLC Company OS. These docs
were migrated from `C:\code\company\docs` so Zero can use them as first-class
context for task planning, company reports, agent routing, finance/legal
readiness, consulting operations, product strategy, and robotics lab tracking.

## Source-of-truth order

1. Runtime truth: Zero app state, Zero database records, live service health,
   and Legion project/sprint status.
2. Company doctrine: `mandate.md`, `architecture.md`, `master-plan.md`.
3. Operating system docs: task, agent, dashboard, finance, second-brain, and
   robotics/product docs in this folder.
4. Evidence and narrative memory: Obsidian weekly reviews and source notes.
5. Legacy archive: `C:\code\company` and `_source-tree/` in this folder.

## Task-management decision

Zero is the canonical task and operating system. Obsidian is the narrative
memory and review mirror. Notion is deferred until a client, contractor, or
external collaboration workflow actually needs it.
