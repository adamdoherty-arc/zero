# WhatsApp Formatting Rules

When formatting responses for WhatsApp delivery, follow these strict rules:

## Syntax
- *Bold*: Single asterisks `*text*` — NEVER double asterisks
- _Italic_: Single underscores `_text_`
- ~Strikethrough~: Tildes `~text~`
- `Monospace`: Triple backticks for code/IDs
- Bullet lists: Single asterisk + space `* Item`
- Numbered lists: `1. Item`
- Quotes: `> text`

## Prohibited (will show raw characters)
- No headers (`#`, `##`, `###`) — use *BOLD CAPS* instead
- No markdown tables — use bullet lists
- No horizontal rules (`---`)
- No nested formatting that risks showing raw symbols

## Goal
"Human-to-Human" look. Technical but clean. Mobile-first reading experience.
